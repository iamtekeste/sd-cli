# sd-cli
> Command to minimize all open windows and show desktop from the terminal. Currently only works on Ubuntu.

## CLI

```
$ npm install --global sd-cli
```

```
$ sd-cli --help

  Usage
    sd-cli

  Examples
    $ sd-cli
```
